## 使用方法
***下载最新的Release，解压后运行 Dial.exe 文件即可，第一次运行如果出现报错——未找到当前拨号连接的号码本入口_623，请尝试使用管理员权限重新启动应用***

3. jre精简比较粗糙，程序压缩包偏大
